Para la correcta ejecución de los códigos de este repositorio, se debe tener instalado en el computador: Visual Studio Code y Python, donde Visual Studio Code requiere tener las siguientes extensiones habilitadas:
pip-packages
pip installer
pip manager

Una vez se tiene esto, se deben ejecutar los siguientes comandos en la terminal:
.venv\Scripts\python -m pip install scikit-learn
pip install imbalanced-learn
pip install xgboost
pip install torch

Y luego para poder ejecutar el proyecto, se debe ejecutar el siguiente comando:
python main.py
